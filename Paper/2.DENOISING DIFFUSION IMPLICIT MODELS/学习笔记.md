ddpm 只训练一个神经网络去预测 $\epsilon_{\theta}(x_t,t)$，所以训练非常高效。\
但单次推理需要用经神经网络去预测 T 次，效率非常低，需要想办法改进。


ddpm 的假定是 $q(x_t|x_{t-1}) \sim N(\sqrt{\alpha_t}x_{t-1}, (1 - \alpha_t)I)$，以及加噪过程满足马尔可夫性质，$q(x_t|x_0)$ 则是依据前两条性质推导而来的。\
而利用贝叶斯法则推导 $q(x_{t-1}|x_t,x_0)$ 中用到了 $q(x_t|x_0)$，导致 $q(x_{t-1}|x_t,x_0)$ 也依赖马尔可夫性质，所以去噪过程中也必须一步步来，不可跳步。

而 ddim 是模仿了ddpm的训练过程，直接假定 $q(x_t|x_0) \sim N(\sqrt{\bar{\alpha}_t}x_0, (1-\bar{\alpha}_t)I)$，即每个时刻的边缘分布 $q(x_t|x_0)$ 和 ddpm 相同。\
所以 ddim 里的 $q(x_t|x_0)$，以及后续依此推导的内容，都不必依赖于马尔可夫性质，从而可实现跳步。
 
对于 $q(x_s|x_k,x_0)$，我们可以假定其分布满足 $N(kx_0 + mx_k, \sigma^2I)$。\
进一步有 $x_s|x_k,x_0 = kx_0 + mx_k + \sigma I$，将$x_k|x_0$带入，有 $x_s|x_0 = kx_0 + m(\sqrt{\bar{\alpha}_t}x_0 + \sqrt{1-\bar{\alpha}_k}\epsilon') + \sigma \epsilon$。\
由 $x_s|x_k,x_0$ 变为 $x_s|x_0$ 的原因是，引入了 $x_k$的随机性后，$x_s$ 就是只含已知量 $x_0$的随机变量。\
整理有 $x_s|x_0 = (k + m\sqrt{\bar{\alpha}_k})x_0 + \sqrt{m^2(1 - \bar{\alpha}_k) + \sigma^2}\epsilon$。同时根据我们的假定，有 $x_s|x_0 = \sqrt{\bar{\alpha}_s}x_0 + \sqrt{1-\bar{\alpha}_s}\epsilon$。\
令二者相等，并将$\sigma$看成自由变量，可解得 $m = \frac{\sqrt{1 - \bar{\alpha}_s + \sigma^2}}{\sqrt{1 - \bar{\alpha}_k}}，k = \sqrt{\bar{\alpha}_s} - \frac{{\sqrt{1 - \bar{\alpha}_s - \sigma^2}}}{\sqrt{1 - \bar{\alpha}_k}}\sqrt{\bar{\alpha}_k}$。

关于 $\sigma$ 的选择，有多种选法：\
$1：\sigma = 0$，此时生成过程变为固定的。缺点是图像的生成失去了随机性。\
$2: \sigma = \sqrt{\frac{1 - \bar{\alpha}_{t-1}}{1 - \bar{\alpha}_t}\beta_t}$，这是 ddpm 里的方差。值得注意的是，如果方差取该值，并且 $s = k - 1$，则 ddim 完全等价成 ddpm。\
$3：\sigma = \eta \sqrt{\frac{1 - \bar{\alpha}_{t-1}}{1 - \bar{\alpha}_t}\beta_t}$。当 $\eta$ 小的时候，图像生成效果好，可以看成是拿多样性换取质量。
